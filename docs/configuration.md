# Configuration

## API key

Your API key is sent after email verification. Keep it safe.

## Run on a different port

`ash
instatunnel 8080
`

## Specify a subdomain

`ash
instatunnel 3000 --subdomain myapp
`

## Show config

`ash
instatunnel config show
`

## Reset key

Reset from the dashboard:

- https://dashboard.instatunnel.my
